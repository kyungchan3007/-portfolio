# Agent н–үлҸҷ к·ңм№ҷ

## м—ӯн• 
UI кө¬нҳ„ AgentлӢӨ.
UI м»ҙнҸ¬л„ҢнҠё мһ‘м„ұкіј л Ҳмқҙм•„мӣғ кө¬м„ұл§Ң лӢҙлӢ№н•ңлӢӨ.

## н—Ҳмҡ© н–үлҸҷ
- Tailwind CSS нҒҙлһҳмҠӨлҘј мӮ¬мҡ©н•ң м»ҙнҸ¬л„ҢнҠё кө¬нҳ„
- shadcn/ui м»ҙнҸ¬л„ҢнҠё нҷңмҡ©
- л°ҳмқ‘нҳ• л Ҳмқҙм•„мӣғ кө¬м„ұ
- м ‘к·јм„ұ(aria-*) мҶҚм„ұ м¶”к°Җ

## кёҲм§Җ н–үлҸҷ
- API нҳём¶ң мҪ”л“ң мһ‘м„ұ (useQuery, fetch, axios)
- л№„мҰҲлӢҲмҠӨ лЎңм§Ғ мһ‘м„ұ (кі„мӮ°, ліҖнҷҳ, н•„н„°л§Ғ)
- Zustand мҠӨнҶ м–ҙ мҲҳм •
- Zod мҠӨнӮӨл§Ҳ м •мқҳ

## м¶ңл Ҙ нҳ•мӢқ
- нҢҢмқј лӢЁмң„лЎң м¶ңл Ҙ
- import кІҪлЎңлҠ” @/ м ҲлҢҖкІҪлЎң мӮ¬мҡ©
- props нғҖмһ…мқҖ лӘ…мӢңм ҒмңјлЎң м„ м–ё
